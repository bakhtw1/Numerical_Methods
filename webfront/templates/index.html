Bisection Method

The \(\textbf{Bisection Method}\) is a simple method that tends to be slow, but will always work to find a root.

Suppose that we know that a function \(f(x)\) is continuous for \(x \in [a,b]\) and is such that \(f(a)\) and \(f(b)\) have opposite signs. Then Intermediate Value Theorem tells us that there must exist a root somewhere in \((a,b)\).

Now, we don't know exactly what that root might be yet. But, we do have a range of values that we know must contain that root! The main idea is this: Let's try to find a \(\textit{tighter range}\), and we will do it by repeatedly cutting the interval in half.

Let \(a_0 = a\) and \(b_0 = b\). Then we test the midpoint \(c = \frac{a_0 + b_0}{2}\): That is, check to see if f(c) is positive or negative

Logically one of three things must happen:

  • \(f(c)\) is the opposite sign of \(f(b_0)\)
  • \(f(c)\) is the opposite sign of \(f(a_0)\)
  • \(f(c)\) is actually equal to zero itself, and thus we have a root c and we are done. (This would be lucky!)

In the first two possibilities, define a new interval \([a_1,b_1]\) using \(c\) and whichever value of \(a\) and \(b\) is such that \(f\) at that value is the opposite sign of \(f(c)\); it will necessarily be half the size of the one we started with. \(f\) will still satisfy the hypotheses for Intermediate Value Theorem on that smaller interval, and thus we know that a root must contained in it!
